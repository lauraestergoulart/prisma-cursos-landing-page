## 🔄 Regras Básicas do Fluxo
1. **Trello:** Escolha uma tarefa no Backlog, coloque seu nome e arraste para *Em Andamento*.
2. **Git/Branches:** Crie uma branch própria para a sua tarefa (`feature/nome-da-pagina`). Nunca programe direto na `main`.
3. **Entregáveis:** Só mova para *Concluído* após marcar todos os itens do checklist do cartão.
4. **Deploy:** Abra um Pull Request (PR) para a `main` e chame alguém do grupo para revisar o código.
